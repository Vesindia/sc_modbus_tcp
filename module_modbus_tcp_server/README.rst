Modbus TCP Server
=================

:scope: General Use
:description: Modbus TCP Server component
:keywords: ModbusTCP
:boards: XA-SK-E100

Features
--------

Implements following Modbus commands received over TCP:

  * Read Coils
  * Write Single Coil
  * Read Input Register
  * Read Holding Registers
  * Write Single Register
  * Read Discrete Inputs
